## Getting Started

1. Clone the repo
   ```
   https://github.com/kim-gunwoo/react-infinite-scroll.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start
   ```sh
   npm start
   ```

## Use

- CRA(create-react-app)
- react-hook
- styled-components
- styled-reset
